1) Введение
Область применения;
Краткое описание возможностей;
Уровень подготовки пользователя;
1.1) Область применения
Защита личных данных от злоумышленников.
1.2) Краткое описание возможностей
Программа используется для защиты данных пользователя.
1.3) Уровень подготовки пользователя
Пользователь должен уметь работать в таких ОС, как MS Windows (XP/7/8/10).
2) Назначение и условия применения игры
Учет транспортных средств логистической или автотранспортной компании.
3) Подготовка к работе
3.1) Порядок загрузки данных и программ
Установщик игры расположен на CD-диске. Для установки программы пользователю требуется запустить «Setup.exe»
3.2) Порядок проверки работоспособности
Для запуска игры на рабочем столе появилась иконка.
Если программа была установлена неверно, следует выполнить повторную установку через «Setup.exe».
В окне помощи «Помощь», показана инструкция использования приложения.
Руководство пользователя находится на CD-диске в файле «README.txt». Данный файл появится в корневой папке UchetAvto после ее установки.
